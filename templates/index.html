Shortcut-ready endpoint: /download?project=<project-name>

Shortcut integration

Use a Shortcut that sends a GET request to this backend. Example:

https://your-domain.com/download?project=my-first-project

The response is a ZIP file containing the selected project folder.

API

  • GET / — website listing
  • GET /projects.json — JSON list of available project names
  • GET /download?project=<name> — downloads a ZIP archive
  • GET /download/<name> — downloads a ZIP archive
